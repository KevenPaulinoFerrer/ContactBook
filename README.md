# Contact Book App

A basic Contact Book app built with React Native featuring CRUD operations, 
offline-first architecture, and mock sync functionality. Tested on Android.

## Prerequisites
- Node.js >= 18
- JDK 17
- Android Studio with an emulator or physical device
- React Native environment set up — follow the [official guide](https://reactnative.dev/docs/environment-setup)

## Install Dependencies
Run this command from the root of the project:
```sh
# Using npm
npm install
# OR using Yarn
yarn install
```

## Build and Run

### Android
```sh
# Using npm
npm run android
# OR using Yarn
yarn android
```

### iOS
> Not tested. Compatibility not guaranteed.
```sh
# Using npm
npm run ios
# OR using Yarn
yarn ios
```
